# DevTrails React Frontend (Vite)

Lightweight React + Tailwind + Recharts app (mocked) for the gig-worker insurance UI.

Run locally:

```bash
cd frontend/react-app
npm install
npm run dev
```

Notes:
- The app uses mock data by default (no backend required). Replace `src/services/api.js` with real endpoints to connect to your backend.
- Routes: `/` `/register` `/risk` `/policy` `/dashboard` `/alerts` `/claims` `/wallet`
