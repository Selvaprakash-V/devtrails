# Next.js frontend scaffold (mock)

This folder contains a lightweight Next.js + Tailwind + Recharts scaffold for the gig-worker parametric insurance UI.

Run (if dependencies installed):

```bash
cd frontend
# install next, react, react-dom, tailwindcss, postcss, autoprefixer, recharts, axios
# example: npm install next react react-dom tailwindcss postcss autoprefixer recharts axios
npm run dev
```

Pages:
- `/register` - Worker onboarding
- `/policy` - Policy activation
- `/dashboard` - Worker dashboard
- `/claims` - Claims list
- `/wallet` - Wallet
- `/admin` - Admin dashboard

API mocks are under `pages/api/*`.
